Windows 提权信息收集

Windows提权信息涉及多个方面，下面分别进行介绍。

1. 操作系统的名称和版本信息

2. 主机名称和所有环境变量

主机名称：hostname
环境变量：SET

3. 查看用户信息

查看所有用户:net user或net1 user
查看管理员用户组:net localgroup Administrators或net1 localgroup Administrators
查看远程终端在线用户:query user或quser

4. 查看远程端口

（1）通过注册表查看
REG query HKLM\SYSTEM\CurrentControlSet\Control\Terminal Server\WinStations\RDP-Tcp/v PortNumber

（2）通过命令行查看
获取对应的PID号：tasklist /svc | find "TermService"
通过PID号查找端口：netstat -ano | find "1980"

5. 查看网络情况

网络配置情况：ipconfig /all
路由器信息：route print
查看ARP缓存：arp -a
查看网络连接：netstat -ano
查看防火墙规则netsh firewall show config或netsh firewall show state
关闭防火墙：netsh advfirewall set publicprofile state offornetsh firewall set opmode disable

6. 应用程序和服务

查看服务的进程ID：tasklist /SVC
查看已安装驱动程序的列表：DRIVERQUERY
查看已经启动的Windows服务：net start
查看某服务的启动权限：sc qc TermService
查看已安装程序的列表：wmic product list brief
查看服务列表：wmic service list brief
查看进程列表：wmic process list brief
查看启动程序列表：wmic startup list brief
查看已安装的更新和安装日期：wmic qfe get Caption,De**ion,HotFixID,InstalledOn
搜索可以提升权限的特定漏洞：wmic qfe get Caption,Description,HotFixID,InstalledOn | findstr /C:"KBxxxxxxx"
结束程序： wmic process where name="iexplore" call terminate

8. 目录文件操作

列出d:\www下的所有目录：for /d %i in (d:\www\*) do @echo %i
把当前路径下文件夹名字只有1~3个字母的显示出来：for /d %i in (???) do @echo %i
以当前目录为搜索路径，把当前目录及其子目录中的全部EXE文件列出：for /r %i in (*.exe) do @echo %i
以制定目录为搜索路径，把当前目录及其子目录中的所有文件列出：for /r "f:\freehost\hmadesign\web\" %i in (*.*) do @echo %i
显示a.txt中的内容，因为/f的作用，会读出a.txt中的内容：for /f %i in (c:\1.txt) do echo %i

从网站上收集信息

  1. 公司或者个人网站是信息的重要来源

同一个企业的员工往往会登录相同的论坛，有着类似的兴趣，甚至会上相同的几个社交网站。
很多员工会在社交网站上用标签的形式展示自己的职位（国内对应的比如：某群，某圈等），这可以令社会工程人员勾勒出公司某个部门的规模以及组织架构。

  1. 搜索引擎

《Google Hacking for Penetration Testers》，这本书介绍了谷歌搜索特定信息的语法，部分语法适应国内的某度。
同时推荐一个网页：www.googleguide.com/advanced_operators.html

  1. Whois域名信息查询

Whois能提供域名数据库查询服务。Whois数据库中有很多有价值的信息，有些时候甚至包括网站管理员的完整联系方式

网页版：www.whois.net
Linux版： whois 命令

  1. 公共服务器

企业对外的公共服务器往往会提供网站所没有的很多信息，比如服务器的操作系统，安装的应用程序和IP地址，这些信息可以大致反应企业的信息服务架构。
IP地址可以说明服务器是在本地还是从服务器提供商处租聘的；通过域名解析记录可以看出服务器的名称、功能、以及IP地址分布。

  1. 社交媒体

很多公司热哀于在社交网站上做推广和营销。这里提供了来自企业的另外一股信息流：活动安排、新产品发布、新闻报道以及一些能与当前热点事件挂钩的文章等等。

  1. 个人网站、博客等

像博客、维基、网络视频等个人网站不仅会提供目标公司的信息，还会透漏这些信息上传者的个人观点和信息。在博客上对企业满腹牢骚的员工会和那些持有类似观点的人相聊甚欢。
很多人都没有意识到智能手机拍摄的照片会隐藏GPS信息，也有很多人喜欢在某圈上分享自己的地理位置、和谁在一起以及正在做的事情等。
只需片刻功夫，目标人物的住址、工作、照片、兴趣等信息就呈现在眼前了
社交网站成为最佳信息源的另一个原因是可以匿名伪装。如果目标人物是一个刚离婚的中年男子，平时热哀与跟新某论坛，那么你就可以假扮成一名希望结交新朋友的年轻女士。很多时候，人们在被拍马屁时，会泄露很多重要信息。结合伪装的技术，再加上人们通常认为自己见到、读到的就是真实信息（人们通常只愿意看到自己想看到的）这一漏洞，便很容易套取更多信息。

  1. 公开报告

公开数据可能来自目标企业内部或者外部，包括季度报告、政府报告、分析报告及公开交易公司的收入信息等。

运用观察的力量

目标企业员工使用钥匙、门禁卡还是其他方式进入办公大楼？
有没有指定的吸烟区？
垃圾桶有没有上锁？
办公大楼有外置摄像头吗？
供电系统或空调机组等外围设备的维修公司是哪家？
这些信息都可以给社会工程人员入侵提供可能。
上面仅仅是通过观察可以得出答案的几个问题而已。花上一段时间观察目标，并用隐藏式摄像机录制下来，然后回去慢慢研究和分析，你会学到很多知识并且你的信息量也会暴增

运用分析软件

Common User Passwords Profiler(“常用用户密码探查器”，缩写为CUPP)和Who’s Your Daddy(“谁是你爸爸”，缩写为WYD)是常用的密码分析工具，社会工程人员可利用它们分析出企业或个人可能使用的密码。
